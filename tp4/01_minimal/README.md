# TP4 - Partie 1 : Exemples minimaux

## Objectif
Cette partie sert d'exemple a lire et a executer.

Elle montre, sur des cas tres petits, la structure generale d'un projet :
- Python
- -> code natif
- -> eventuellement CUDA
- -> retour vers Python

## Statut pedagogique
Exemple.

Il ne s'agit pas du coeur de l'evaluation.
Le but est surtout d'observer, executer, puis reutiliser cette structure comme modele.
